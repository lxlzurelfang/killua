Cara Pakai:
1. Pastikan Python 3 sudah terinstall.
2. Install modules jika perlu:
   pip install requests

3. File yang harus ada:
   - scriptcrawler.py
   - keywords.json

4. Jalankan script:
   python3 scriptcrawler.py

Keterangan:
- Auto load keyword dari keywords.json
- 10 keyword random tiap cycle
- Multi-threading scraping
- Sleep 15 menit setiap cycle
